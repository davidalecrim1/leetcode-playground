# Coin Change Problem

## Approaches
- Pure Brute Force
  - For Loops to try every option.
- Greedy
  - Sort and start at the end.
- Depth For Search
  - Decision tree
  - Recursion
  - Find subproblems
  - Time to memoization
- Dynamic Programming
  - Save the steps using a dp

## Explanation
- dp[amount] = minimum number of coins needed.
